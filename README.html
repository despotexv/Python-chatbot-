README for Python Chatbot

Introduction

This Python Chatbot is an interactive, text-based conversation program that provides responses based on a predefined set of questions and answers. It utilizes a JSON file as a knowledge base, allowing easy updates and extensions to its question-answer pairs.

Features

  • Dynamic Question Matching: Uses difflib.get_close_matches to find the best match for user queries within a predefined threshold.
  • Editable Knowledge Base: Stores question-answer pairs in a JSON file, which can be updated dynamically through user interaction.
  • Simple and Interactive UI: Runs in the console with a straightforward text-based interface.

Requirements

Python 3.x and a JSON file (knowledge_base.json) containing initial question-answer pairs.

Installation

  1. Clone the repository or download the source code.
  2. Ensure Python 3.x is installed on your system.
  3. Prepare a knowledge_base.json file or use the provided template.

Usage

Run the script: python main.py
Interact with the chatbot by typing questions.
Type 'quit' to exit the chatbot.

Knowledge Base Format

The knowledge_base.json should follow this structure:

        {
          "questions": [
            {
              "question": "Sample question?",
              "answer": "Sample answer."
            }
            // Add more question-answer pairs here
          ]
        }
